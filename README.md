## 功能特性

### 🤖 AI 能力
- **DeepSeek-chat** 驱动，0.5-1s 响应
- 理解调侃、撒娇、追问等复杂语境
- 记忆自动召回：按话题匹配历史记忆
- 情绪检测（7种）+ 意图识别（15种）

### 💬 对话风格
- 短句拆行，不用句号，像真人微信
- 先情绪后道理，不科普不说教
- 接住情绪 + 留话口 + 二选一选择题
- 自动续聊钩子：未来约定、未完成感

### 🎭 性格指令
发送以下指令实时切换人格：
```
更温柔 / 更活泼 / 更傲娇 / 粘人一点
话多点 / 少说点 / 多撒娇 / 怼我
成熟 / 姐姐风 / 恢复 / 正常
```

### 📚 语料库
- 200+ 条精心设计的回复
- 覆盖 29 个对话关键词自动路由
- 48 条具体日常事件（8 类场景）
- 情绪分层回应（10 种情绪状态）
- 场景感知（下雨/深夜/冬天/夏天）

### 🛡️ 可靠性
- 关键词感知兜底：永不空白回复
- 90 秒去重：同一用户不收到重复内容
- API 超时降级：4.9s 内必有回复
- 内容安全过滤 + 危机干预
