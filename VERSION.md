# 版本规则 (Semantic Versioning)

## 递增规则

| 变更 | 版本号 | 示例 |
|------|--------|------|
| Bug 修复 / 小调整 | **PATCH** +0.0.1 | 修键名、去重、改参数 |
| 新功能（向下兼容）| **MINOR** +0.1.0 | 新模块、新语料、节奏控制 |
| 架构变更 / 破坏性更新 | **MAJOR** +1.0.0 | 换模型、重写 Prompt 架构 |
